# 准备好去奥林匹克公园赏秋叶了吗？

器材：Canon EOS 5D IV
镜头：EF24-105mm f/4L IS USM、小痰盂
拍摄地点：北京奥林匹克

  初秋的树叶只是有些微黄，但已经开始预示着将要飘零，也预示着故事有些感伤，树树秋声，山山寒色，这声这色，都埋藏着埋藏着一点凄凉，一点悲伤，一点宁静，一点幻想。

  北京进入红叶的最佳观赏期，其中香山红叶的名气最大，自然是人满为患。前两年去过几次香山，风景的确不错，但是不喜人多的我今年决定去北京奥林匹克森林公园走一遭。平日去过不少次奥森，但多是春季，秋季还未去过。此次看后确有古人“自古逢秋悲寂寥，我言秋日胜春朝。”的感叹。

  金秋的阳光温馨恬静，微风和煦轻柔；午后的小路上，洒满了阳光。曲径通幽，禁不住向前漫步。工作了一周，周末难得遇到如此宝地，相对香山等风景名胜之地，倒是多了一分清闲，疲惫不知不觉间已然消散。

  叶子一片片落下，带着一丝丝的遗憾，投向大地母亲的怀抱。她们跳跃着，旋转着，轻舞飞扬着，翩然落下。长椅上，一对小情侣在温暖的阳光中欣赏着此情此景，充满了幸福，爱情让人忘记了时间。

  “潮落江平未有风，扁舟共济与君同”，泛舟湖上赏秋也是别有一番风味。周围满是芦苇，芦花随着微风摆动，听着船桨溅起水花的声音，小船慢慢悠悠消失在视野，只有水面的波纹证明它曾经来过。

  抬头仰望,天空瓦蓝瓦蓝的,洁净得好像刚洗过的蓝宝石,使人感到秋高气爽。秋天是个怀旧的季节，很多的故事和人都在不经意中被勾连，悄悄的说声想念，似乎总是在这样一个凉气渐深的秋天，蓦然回首，发觉自己走过的路竟是那般苍凉。风花雪月的往事，不堪回首，辛苦奔波的岁月，历历在目。

  秋风吹过，秋叶纷纷，叶子一片一片飘落在地上，带着秋天独有的魅力。渲染着大地的金黄。清凉的空气，金黄色的树叶，这是秋天独有的景色。秋天是沉甸甸的季节，秋天是收获的季节。秋天，写着收获，秋天，写着相思。

  秋天的风，秋天的雨，秋天的叶。它们任何一着都不能独自显示出美丽。它们表达出的，是一种整体的美，一种和谐的美。没有风，雨不会飞，叶不会舞；没有雨，风不会湿润，叶不会害羞；没有叶，风显得单调，雨显得乏味。只有它们巧妙地结合起来，才会构成一幅美丽的画，一首精美的诗--秋。
